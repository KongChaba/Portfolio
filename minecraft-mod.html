Flower Mod

author: Applesss

Minecraft mod about flowers in the world climate has many flowers in almost every biome (no snow biome). Flowers can be used for cooking and many other things to collect.

How To Download

  • Step1 : Installing Minecraft Forge

    1. Navigate to the Minecraft Forge Download
    2. Within the left-hand menu, select the version you require.
    3. Select either Installer from either the Latest or Recommended release.
    4. Save and run the downloaded Forge .jar file.
    5. Check that the information looks correct, then press Ok to install.
    6. Once installed, return to the Minecraft launcher and ensure your Forge version is available.

  • Step2 : Installing Mod

    Click Download here

  • step 3 : Adding Mod

    Click on the search bar, type "%appdata%" then go to minecraft > mod and put the mod file there.
